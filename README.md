# OUTDOORKA 揪好咖 Backend API Project

## API

```http
https://
```

## 參考 `example.env` 建立 `config.env`

```text
PORT=
DATABASE_URL=
DATABASE_PASSWORD=
JWT_TOKEN=
JWT_SECRET=
```

## Setup

Node 版本為 `v20.12.0`

```bash
npm install
```

## Develop

```bash
npm run dev
```

## ESlint Start

```bash
npm run lint
```

## Swagger

SwaggerUI ： <http://localhost:3006/api-docs>

```bash
npm run swagger
```

## 使用的工具套件

- Node.js
- Express
- Typescript
- MongoDB

- Redis
